# ASTHAproject1
DecoBot is a rule-based AI chatbot built using Python. It takes user input, sanitizes it, and matches it to predefined responses using a dictionary. It features a graceful exit command and a fallback response for unknown inputs.
 # Features
🗣️ Interactive Conversation — Real-time terminal-based chat interface
🔍 Keyword Matching — Matches user input to predefined responses
🧹 Input Sanitization — Handles uppercase, lowercase, and extra spaces
🚪 Exit Command — Graceful shutdown with exit keyword
⚡ Lightweight — Zero dependencies, runs on pure Python
🛡️ Fallback Response — Handles unknown inputs gracefully

# Tech Stack
Technology         Usage
Python 3.x         Core programming language
Dictionary (dict)  Response mapping / rule engine
input() / print()  Terminal I/O
String methods     nput sanitization (.lower(), .strip())
